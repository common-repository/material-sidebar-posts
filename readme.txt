Provides latest posts with material style design, Display the posts with image and no thumbnail style and much more. 

== Description ==

This plugin will have options to display your recent posts with material style design, you can display your latest posts via widget.it will allow you to display list of latest posts with thumbnail image, publish date, no thumbnail design and much more.

= More Features =

*   Customizable widget title.
*   Option for custom post type 
*   Automaticaly date calculate from the publish date to current date.
*   Limit the posts list via widget.
*   Display your image with circle and square.
*   Allow you to multiple style post display design in your sidebar.
*   Display content with customizable length for both post title and content.
*   Display thumbnail with customizable width and height and alignment.
*   Custom background color for the title box.
*   Order by ID, Title, Date, Random.

This plugin is good to use for the blog sidebar and website sidebar of your webpage.you can change everything via widget options

== Installation ==

**Through Dashboard**

1. Login to your wordpress admin area and go to plugins menu and click to add new plugin sub menu.
2. Type **Material Sidebar Posts** in the plugin page search box and hit enter.
3. Find Material Sidebar Posts plugin
4. Click the install now button and activate the plugin right way.
5. Go to Widget page and find the **Material Sidebar Posts**
6. Drag and Drop to your widget area to display.

**Through Ftp**

1. Download the plugin to your computer.
2. Unzip the plugin zip file.
3. Upload the material-sidebar-posts folder in to the plugin directory via ftp or filezilla.
4. Login to your admin dashboard and click on the plugin menu.
5. Activate the plugin.
